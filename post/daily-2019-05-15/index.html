空值区分，单双引号，读取文件

null,0,'',false,‘0’的区别，如何判断

  • empty操作以上五个变量，都返回false
  • null为不存在之意：php底层的zval空间里没有存其value值，只存储了一个type标志其 IS_NULL(所以解释了 empty(null)=true，isset(null)=false ，isset('')=true)
  • 其他四个为存在，php底层是开辟zval空间存储，有value,有type
  • string '0'与 string '' 不相等
  • int 0 却和 string '' 相等
  • string '0' 与 null 不相等
  • int 0 与 null 相等

PHP中单引号和双引号的区别

  • 单引号的运行速度会比较快，双引号会比较慢，原因在于双引号要先查找语句中是否有变量，而单引号则不用
  • 双引号内的变量会解释出值，但是在单引号内则不做处理，直接输出
  • 单引号：只转义单引号 ” 和 \ ;
  • 双引号：可以转义大量的特殊符号

PHP读取文件内容的方法

    $file_path = "test.txt";
    $fp = fopen($file_path,"r"); 
    $str = fread($fp,filesize($file_path));

    $file_path = "test.txt";
    $str = file_get_contents($file_path);
